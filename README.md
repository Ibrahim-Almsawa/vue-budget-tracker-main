## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/EbrahimMusawa/vue-budget-tracker.git
   ```

2. Install dependencies:
   ```bash
   cd vue-budget-tracker
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   ```
